The process

  • look through big task - define problems or obstacles and possible solutions

  • creating wireframes and user flows
  • estimate time

Execution

Even if we use Foundation framework for front-end with main grid system, some basic building blocks and few plugins, still most of the styling elements are custom. The biggest challenges for me was working with templates and create reusable design blocks which we can be applied across site. Our platform has complicated content structure, including different layers, connections, different views for different type of users and pages generated by blade templates with php loops inside etc. As a front-end developer I was responsible for applying design taking all this factors in consideration, making sure that different pages look similar, independent from amount and type of content on each page. At the same time I was making sure that our web-site is responsive. For new web design guidelines I was creating building blocks with basic styling that we can use across website. It was important for me to create “wrappers” that can cover most of the cases in terms of what content they should wrap.
